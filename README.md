# Road-Rail Freight Digital Twin

University of Glasgow Summer Project

Goal:
Build a digital twin that compares
naive and connectivity-aware freight
coordination strategies and measures
their impact on carbon emissions.

## Project Vision

This project will model a road-rail freight system
using a digital twin.

The simulation will compare:

1. A naive twin that assumes all information is current.

2. An intelligent twin that accounts for
connectivity gaps and information age.

The goal is to determine how much carbon can
be saved through better decision-making.
